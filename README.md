# GamerVibe – Unified Gaming Information & Blog Application

A comprehensive **web and desktop application** designed for gamers seeking an all-in-one platform to explore game details, read/write reviews, and manage blogs.  
The system integrates a **C# ASP.NET web application** and a **Windows Forms desktop client**, delivering a seamless user experience across platforms.

---

## Features
- Full **CRUD functionalities** for games, blogs, reviews, and users.  
- Integrated **authentication and role-based access control**.  
- **Layered architecture** (Presentation, Business Logic, Data Access) ensuring maintainability and scalability.  
- Applied **SOLID principles** and **design patterns** for clean and testable code.  
- Developed **responsive UIs** for both web and desktop versions.  

---

## Tech Stack
**Languages & Frameworks:** C#, ASP.NET, WinForms  
**Database:** Microsoft SQL Server  
**Tools:** GitLab, Visual Studio  
**Concepts:** OOP, SOLID Principles, Design Patterns, CRUD, Role-Based Access

---

## Setup Instructions
1. Clone this repository.  
2. Add your own `appsettings.json` file (add file to the WebApp and DALClassLibrary).

  **Example structure:**

{
  "ConnectionStrings": {
    "DefaultConnection": "Server=YOUR_SERVER; Database=YOUR_DATABASE; User Id=YOUR_USER; Password=YOUR_PASSWORD; TrustServerCertificate=True"
  }
}

3. Run the ASP.NET project in Visual Studio.  
4. Launch the WinForms client to access features locally.
